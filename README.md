# Lingo Asset Fetcher CLI

> Automate asset retrieval from lingoapp.com

## Install

```
npm install laf -g
```

## Usage

#### Generate Config File

> Config file is required. Using generator to create it is optional

1. Generate configuration file
   \$ `laf gen`

2. Fetch files
   \$ `laf fetch`
